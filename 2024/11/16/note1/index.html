我的个人博客

2024-11-16 笔记 博客

在初步调研之后，选择了Hexo来搭建我的个人博客，主要原因是发现了一个很好看的博客主题hhh

基础Hexo

  1. Git安装：之前安装过了，这里暂且不表

  2. Node.js安装：Node.js安装及环境配置之Windows篇 - 刘奇云 - 博客园

    我的Node.js装在了C盘，npm安装的全局模块所在的路径，以及缓存cache的路径则在”D:\node”

  3. Hexo安装：npm install hexo-cli -g

  4. Github:

    • Create a new repository:仓库名为1900.github.io
    • 桌面右击打开Git bash，输入ssh-keygen -t rsa -C "邮件地址"
    • 进入C:\Users\用户名，在里面进入.ssh文件用记事本打开里面的id_rsa.pub,全选复制里面的代码
    • 在github中进入用户设置，找到SSH keys新建SSH keys，粘贴代码然后创建，在git bash中输入ssh -T git@github.com回车后输入yes检查是否成功

  5. Blog：

    • 新建文件Blog，进入文件夹打开gitbash运行hexo init myblog,进入myblog，hexo install，hexo g，hexo s，便生成了基础的hexo界面
